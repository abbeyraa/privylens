# OtoMate

OtoMate is a web automation builder for internal workflows. It lets you design step-by-step flows, inspect pages in a real browser, and run the automation in sequence with a clear execution report.

## Main Features

- Flow editor with sub‑menus and step ordering
- Inspector that records clicks into JSON logs
- Runner that executes steps in a real browser and stops on failure
- Action types: click, input, wait, navigate
